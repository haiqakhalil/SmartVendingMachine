# 🏧 Smart Vending Machine System

A web-based Vending Machine application built with Java Spring Boot, Next.js, and MySQL.

## 🛠 Tech Stack
- Java 17+
- Spring Boot 3.3.5
- Next.js 15 (Frontend)
- MySQL 8
- Maven
- GitHub

## 📁 Project Structure
- `backend/` → Java Spring Boot (MVC)
- `frontend/` → Next.js UI
- `database/` → MySQL schema
- `docs/` → Documentation
